**OtpInput component (src/components/onboarding/OtpInput.js):**

Create a `'use client'` component. Named export `OtpInput`. Props: `{ onComplete, disabled }`.

- 6 individual `<input>` elements, each `type="text"` with `inputMode="numeric"` and `maxLength={1}`
- Each input: `className="size-11 text-center text-xl font-semibold border border-stone-200 rounded-lg focus:outline-none focus:ring-2 focus:ring-[#C2410C] focus:ring-offset-1"`
- Each input: `aria-label={`Digit ${i + 1} of 6`}`
- Wrap all inputs in `<div role="group" aria-label="Email verification code" className="flex gap-2 justify-center">`
- Use `useRef([])` to hold refs to all 6 inputs
- On change: if value entered and index < 5, auto-focus next input
- On keydown: if Backspace on empty input and index > 0, focus previous input
- On paste: distribute pasted characters across inputs, auto-complete if 6 chars
- When all 6 digits filled, call `onComplete(code)` with the joined string
- Auto-focus first input on mount via useEffect

2. On success, `router.push('/onboarding/profile')` (middleware will redirect to dashboard if already complete)
