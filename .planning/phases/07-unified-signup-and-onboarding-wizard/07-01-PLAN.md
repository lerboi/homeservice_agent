**Middleware rewrite (src/middleware.js):**

Replace the entire middleware with new routing logic. The key changes:

1. Define `AUTH_REQUIRED_PATHS` array: `['/onboarding/profile', '/onboarding/services', '/onboarding/contact', '/onboarding/test-call', '/dashboard']`
2. `/onboarding` (step 1) is PUBLIC — no auth required (it IS the auth step)
3. Three routing cases:
   - Unauthenticated user on AUTH_REQUIRED_PATHS → redirect to `/onboarding`
   - Authenticated user on `/onboarding` → check `onboarding_complete` from tenants table:
     - If `true` → redirect to `/dashboard`
     - If `false` or no tenant → allow through (user needs to complete wizard)
   - Authenticated user on onboarding sub-paths (profile/services/contact/test-call) → check `onboarding_complete`:
     - If `true` → redirect to `/dashboard` (WIZARD-07)
     - If `false` → allow through
   - `/dashboard` paths → require auth only (no DB query for onboarding_complete on dashboard — avoid latency per Pitfall 1 in RESEARCH.md)
4. Keep the existing Supabase cookie handling pattern (createServerClient with getAll/setAll)
5. Matcher config: `['/onboarding/:path*', '/dashboard/:path*']`

Important: The DB query for `onboarding_complete` must ONLY run on `/onboarding` paths, NOT on `/dashboard` paths. Dashboard routes only need the auth check.

```javascript
const AUTH_REQUIRED_PATHS = [
  '/onboarding/profile',
  '/onboarding/services',
  '/onboarding/contact',
  '/onboarding/test-call',
  '/dashboard',
];
```

For the onboarding_complete check, query: `supabase.from('tenants').select('onboarding_complete').eq('owner_id', user.id).single()`

**useWizardSession hook (src/hooks/useWizardSession.js):**

Create a new file. This is a `'use client'` hook. Key prefix per UI-SPEC: `gsd_onboarding_`.

```javascript
'use client';
import { useEffect, useState } from 'react';

export function useWizardSession(key, defaultValue) {
  const storageKey = `gsd_onboarding_${key}`;

  const [value, setValue] = useState(() => {
    if (typeof window === 'undefined') return defaultValue;
    try {
      const stored = sessionStorage.getItem(storageKey);
      return stored ? JSON.parse(stored) : defaultValue;
    } catch {
      return defaultValue;
    }
  });

  useEffect(() => {
    try {
      sessionStorage.setItem(storageKey, JSON.stringify(value));
    } catch {
      // sessionStorage unavailable — degrade gracefully
    }
  }, [value, storageKey]);

  return [value, setValue];
}

export function clearWizardSession() {
  if (typeof window === 'undefined') return;
  try {
    const keys = [];
    for (let i = 0; i < sessionStorage.length; i++) {
      const k = sessionStorage.key(i);
      if (k && k.startsWith('gsd_onboarding_')) keys.push(k);
    }
    keys.forEach((k) => sessionStorage.removeItem(k));
  } catch {
    // silent
  }
}
```

The hook returns `[value, setValue]` like useState. The `clearWizardSession` export is called on wizard completion.
