## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding |
| sm | 8px | Intra-element spacing (icon + label, pill + text) |
| md | 16px | Default element spacing, card internal padding |
| lg | 24px | Section internal padding, feature grid gap |
| xl | 32px | Card gaps, nav height |
| 2xl | 48px | Section vertical padding (mobile) |
| 3xl | 64px | Section vertical padding (desktop), hero vertical breathing room |

Exceptions:
- Touch targets (nav CTA button, mobile menu) minimum 44px height per WCAG 2.5.5
- Hero section top padding: 96px desktop / 64px mobile to clear sticky nav

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (regular) | 1.5 |
| Label / Caption | 14px | 400 (regular) | 1.5 |
| Heading | 24px | 600 (semibold) | 1.2 |
| Display | 48px desktop / 36px mobile | 700 (bold) | 1.1 |

Notes:
- Feature grid card titles: 20px / weight 600 / line-height 1.3
- Testimonial quote text: 16px / weight 400 / line-height 1.6 (slightly looser for readability)
- Nav link labels: 14px / weight 500
- Only 2 font weights in active use for body/heading system: 400 and 600. Bold (700) reserved for the single display headline only.
- Font family: `var(--font-inter), system-ui, sans-serif` (established in `globals.css`)

Source: Defaults applied; discretion granted in CONTEXT.md for typography choices within palette.

## Interaction Contract

| Interaction | Behavior |
|-------------|----------|
| Sticky nav scroll | Nav background transitions from transparent to `--color-landing-dark` with `backdrop-blur` after 20px scroll |
| Nav CTA | Smooth-scrolls to Final CTA section (anchor `#cta`) |
| "Watch Demo" | Links to `/demo` placeholder route — 404 or coming-soon page acceptable in this phase |
| Primary CTA (all 3) | `href="/onboarding"` — navigates to existing onboarding wizard |
| Section entrance animation | Each section fades in + slides up 24px over 400ms, easing `ease-out`, triggered when section enters viewport (Framer Motion `whileInView`) |
| Animation stagger | Feature grid cards stagger 80ms apart. Testimonial cards stagger 80ms apart. |
| Mobile nav | Sticky bar with logo left, CTA button right only. No hamburger menu in this phase. |
| Responsive breakpoints | Mobile: <768px (1-column). Tablet: 768px–1023px (2-column grid). Desktop: ≥1024px (2-column feature grid, 3-column testimonials). |
