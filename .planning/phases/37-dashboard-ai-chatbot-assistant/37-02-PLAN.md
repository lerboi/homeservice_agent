    **ChatMessage.jsx** — Create a 'use client' component that renders a single chat message bubble.

    Props: `role` ('user' | 'ai'), `content` (string), `onNavigate` (function — passed to ChatNavLink)

    **Link parsing function** (inside the file, not exported — or export for testing):
    ```javascript
    export function parseMessageContent(content) {
      const linkRegex = /\[([^\]]+)\]\((\/dashboard[^)]+)\)/g;
      const links = [];
      let match;
      while ((match = linkRegex.exec(content)) !== null) {
        links.push({ label: match[1], href: match[2] });
      }
      const text = content.replace(linkRegex, '').trim();
      return { text, links };
    }
    ```

    **User message rendering:**
    - Row: `flex justify-end`
    - Bubble: `bg-[#C2410C] text-white rounded-2xl rounded-br-sm px-4 py-2 max-w-[80%] text-sm`

    **ChatbotSheet.jsx** — Create a 'use client' component that is the root chat panel.

    Props: `open` (boolean), `onOpenChange` (function), `currentRoute` (string)

    **State:**
    - `messages` — array of `{ id, role, content }` — initialized with the greeting message:
      ```javascript
      const GREETING = {
        id: 'greeting',
        role: 'ai',
        content: "Hi, I'm Voco AI. I can help you navigate the dashboard, understand your data, or answer questions about how features work. What would you like to know?",
      };
      const [messages, setMessages] = useState([GREETING]);
      ```
    - `input` — string, controlled input field
    - `isLoading` — boolean, true while awaiting Groq response

    **Refs:**
    - `messagesEndRef` — ref attached to an empty div at the bottom of the message list, used for auto-scroll
    - `inputRef` — ref attached to Input element for autoFocus on sheet open

    **Auto-scroll effect:**
    ```javascript
    useEffect(() => {
      messagesEndRef.current?.scrollIntoView({ behavior: 'smooth' });
    }, [messages.length, isLoading]);
    ```

    **AutoFocus effect** — when `open` changes to true, focus the input:
    ```javascript
    useEffect(() => {
      if (open) {
        setTimeout(() => inputRef.current?.focus(), 100);
      }
    }, [open]);
    ```

    **handleSend function:**
    1. Trim input — if empty, return
    2. Create user message object with `id: Date.now()`, `role: 'user'`, `content: input.trim()`
    3. Append to messages, clear input, set isLoading=true
    4. Build history array: `messages.slice(-10).map(m => ({ role: m.role === 'ai' ? 'assistant' : 'user', content: m.content }))`
    5. `fetch('/api/chat', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ message: userMessage.content, currentRoute, history }) })`
    6. Parse response JSON — if `res.ok`, create AI message from `data.reply`; if not ok, create AI message with text "Something went wrong on my end. Please try again in a moment."
    7. Append AI message to messages, set isLoading=false
    8. Wrap fetch in try/catch — catch block creates error AI message with the same fallback text, sets isLoading=false

    **handleKeyDown:** If `e.key === 'Enter'` and NOT loading, call `handleSend()`

    **handleNavigate:** Call `onOpenChange(false)` to close the sheet when a nav link is clicked

    **Render structure — responsive Sheet:**
    Use a media query approach to determine side. Since this is a client component, use a `useMediaQuery` custom hook or simpler approach:
    ```javascript
    const [isMobile, setIsMobile] = useState(false);
    useEffect(() => {
      const mq = window.matchMedia('(max-width: 1023px)');
      setIsMobile(mq.matches);
      const handler = (e) => setIsMobile(e.matches);
      mq.addEventListener('change', handler);
      return () => mq.removeEventListener('change', handler);
    }, []);
    ```

    Then render:
    ```jsx
    <Sheet open={open} onOpenChange={onOpenChange}>
      <SheetContent
        side={isMobile ? 'bottom' : 'right'}
        className={isMobile
          ? 'max-h-[85vh] rounded-t-2xl p-0 flex flex-col'
          : 'w-[400px] sm:max-w-[400px] p-0 flex flex-col'
        }
        aria-label="Voco AI chat assistant"
      >
        {/* Drag handle — mobile only */}
        {isMobile && (
          <div className="flex justify-center pt-3 pb-0">
            <div className="w-8 h-1 rounded-full bg-stone-300" />
          </div>
        )}

        {/* Header */}
        <SheetHeader className="px-4 py-3 border-b border-stone-100 shrink-0">
          <div className="flex items-center gap-2">
            <MessageSquare className="h-5 w-5 text-[#C2410C]" />
            <SheetTitle className="text-base font-semibold">Voco AI</SheetTitle>
            <Badge variant="secondary" className="text-xs">Help & Navigation</Badge>
          </div>
        </SheetHeader>

        {/* Message list */}
        <div className="flex-1 overflow-y-auto px-4 py-4 space-y-3" aria-live="polite">
          {messages.map((msg) => (
            <ChatMessage
              key={msg.id}
              role={msg.role}
              content={msg.content}
              onNavigate={handleNavigate}
            />
          ))}
          {isLoading && <TypingIndicator />}
          <div ref={messagesEndRef} />
        </div>

        {/* Input row */}
        <div className="px-4 pb-4 pt-2 border-t border-stone-100 shrink-0">
          <div className="flex items-center gap-2">
            <Input
              ref={inputRef}
              value={input}
              onChange={(e) => setInput(e.target.value)}
              onKeyDown={handleKeyDown}
              placeholder="Ask a question..."
              maxLength={500}
              disabled={isLoading}
              className="flex-1"
            />
            <Button
              onClick={handleSend}
              disabled={isLoading || !input.trim()}
              size="icon"
              className="bg-[#C2410C] hover:bg-[#C2410C]/90 text-white h-10 w-10 shrink-0"
            >
              <Send className="h-4 w-4" />
              <span className="sr-only">Send message</span>
            </Button>
          </div>
        </div>
      </SheetContent>
    </Sheet>
    ```
