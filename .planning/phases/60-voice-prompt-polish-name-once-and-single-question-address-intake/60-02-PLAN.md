**Canonical state+directive format** (from RESEARCH.md Pattern 3 + SKILL.md §1 L10):
```
"STATE: <snake_case_code> [| key=value ...] | DIRECTIVE: <imperative-to-the-model>"
```
- `STATE:` is machine-facing; a code the model can branch on.
- Optional key=value pairs carry structured facts (e.g., `appointment_id=abc123`).
- `DIRECTIVE:` tells the model what to DO NEXT. Always phrased in imperative voice AT the model, never at the caller. Must NOT be speakable English.
