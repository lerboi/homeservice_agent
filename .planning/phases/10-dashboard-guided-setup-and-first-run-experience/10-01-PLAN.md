**API route (src/app/api/setup-checklist/route.js):**

Create route with two handlers:

**GET handler:**
1. Import `createSupabaseServer` from `@/lib/supabase-server` and `supabase` from `@/lib/supabase`
2. Authenticate user via `createSupabaseServer()` -> `auth.getUser()`, return 401 if no user
3. Fetch tenant via service-role supabase: `supabase.from('tenants').select('id, business_name, working_hours, onboarding_complete, retell_phone_number, setup_checklist_dismissed').eq('owner_id', user.id).single()` — return 404 if no tenant
4. Run parallel queries via `Promise.allSettled`:
   - `supabase.from('services').select('id', { count: 'exact', head: true }).eq('tenant_id', tenant.id).eq('is_active', true)`
   - `supabase.from('calendar_credentials').select('id').eq('tenant_id', tenant.id).eq('provider', 'google').maybeSingle()`
5. Derive checklist items array with this exact shape:
```javascript
const items = [
  { id: 'create_account', label: 'Create account', complete: true, locked: true },
  { id: 'setup_profile', label: 'Set up business profile', complete: !!tenant.business_name, locked: true },
  { id: 'configure_services', label: 'Configure services', complete: serviceCount > 0, locked: true },
  { id: 'connect_calendar', label: 'Connect Google Calendar', complete: calendarConnected, locked: false, href: '/dashboard/settings#calendar' },
  { id: 'configure_hours', label: 'Configure working hours', complete: !!tenant.working_hours, locked: false, href: '/dashboard/settings#hours' },
  { id: 'make_test_call', label: 'Make a test call', complete: !!tenant.onboarding_complete, locked: false, href: '/dashboard/settings#ai' },
];
```
6. Return `Response.json({ items, dismissed: tenant.setup_checklist_dismissed ?? false, completedCount: items.filter(i => i.complete).length })`

**PATCH handler:**
1. Same auth check as GET
2. Read body: `const { dismissed } = await request.json()`
3. Update: `supabase.from('tenants').update({ setup_checklist_dismissed: !!dismissed }).eq('owner_id', user.id)`
4. Return `Response.json({ ok: true })`
