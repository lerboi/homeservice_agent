## Spacing Scale

Declared values (multiples of 4 — 8-point grid):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon-to-text gaps, badge inline padding |
| sm | 8px | Stacked list item gaps (feature list `space-y-2.5` ≈ 10px, nearest token is 8px) |
| md | 16px | Card internal padding, form element spacing |
| lg | 24px | Intra-section gaps, card header bottom padding |
| xl | 32px | Between tier cards and billing toggle |
| 2xl | 48px | Trial banner vertical padding |
| 3xl | 64px | Hero section top padding (pt-28 = 112px uses Tailwind scale; section breathing room) |

Exceptions:
- Touch targets: all interactive elements (CTA buttons, billing toggle buttons, accordion triggers) maintain `min-h-[44px]` regardless of visual size — this overrides the spacing scale for tap target compliance.
- Hero section top padding is pt-28 (112px) to clear the fixed nav — intentional exception, not a design token change.

Source: Existing PricingTiers.jsx and page.js patterns; preserved from Phase 13 design decisions.

## Interaction Contract

### Billing Toggle

- State: `useState('annual')` — defaults to annual (shows savings)
- Annual selected: price shows discounted rate (monthlyPrice × 0.8), original price struck through
- Monthly selected: price shows full monthlyPrice
- Toggle changes price display in all 4 tier cards simultaneously
- Enterprise always shows "Custom" regardless of toggle state
- Transition: `transition-all duration-150` on toggle buttons

### Tier Card Hover

- Transform: `hover:-translate-y-0.5` (2px lift)
- Border: transitions from `border-white/[0.06]` to `border-[rgba(249,115,22,0.3)]`
- Shadow: `hover:shadow-[0_0_20px_rgba(249,115,22,0.15)]`
- Transition: `transition-all duration-200`
- Touch devices: hover effects disabled via `@media (hover: hover)` — cards remain static on tap

### FAQ Accordion

- One item open at a time (`type="single" collapsible`)
- Open animation: `data-[state=open]:animate-accordion-down` (200ms ease-out, already in globals.css)
- Close animation: `data-[state=closed]:animate-accordion-up` (200ms ease-out)
- ChevronDown rotates 180deg on open: `group-data-[state=open]:rotate-180 transition-transform duration-200`
- Min touch target on trigger: `min-h-[44px]`

### Scroll Animations (AnimatedSection / AnimatedStagger / AnimatedItem)

- Trigger: IntersectionObserver, threshold 0.15, once
- Default variant: fade-up (translateY 20px → 0, opacity 0 → 1)
- Duration: 0.5s ease-out per item
- Stagger delay between AnimatedItems: 0.08s (cards), 0.1s (testimonials)
- Reduced motion: `prefers-reduced-motion` detected by AnimatedSection via `useReducedMotion()` — animations skipped entirely, elements render at final position

### Enterprise CTA — Contact Pre-selection

- Enterprise "Contact Us" button links to `/contact?type=sales`
- Contact page reads `?type` query param and pre-selects the inquiry type dropdown
- No JS required on the pricing page side — plain href with query string
