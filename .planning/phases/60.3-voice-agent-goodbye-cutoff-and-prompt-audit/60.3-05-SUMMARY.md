## Decisions Made

- **Header translation: TERMINAR vs FINALIZAR.** Picked "TERMINAR LA LLAMADA — REGLA CRÍTICA". TERMINAR is the colloquial conversational verb matching the EN "ending" register; FINALIZAR is more formal/bureaucratic. Test invariant asserts the TERMINAR form directly. Documented in the Plan 1 RED commit so future re-reviewers know the alternative was considered.
- **Body phrase style: "Usted" (formal 2nd-person) vs "Tú" (informal).** Picked USTED to match the Spanish info_gathering block already in `prompt.py` (line 305-338 uses "Usted" / "su" consistently). Preserves in-prompt locale register parity.
- **Signature default = "en".** Allows any stale caller to remain functional without touching more files than necessary. Only `build_system_prompt` is updated.
- **"Voco" brand preserved in both examples.** CLAUDE.md mandates Voco; test invariant asserts "HomeService AI" is absent in both locales to prevent future regression.
- **Numerals (9, 10) used for duration-bound assertions, not "minutes"/"minutos".** Avoids accidentally tightening the test in a way that would break if we adjust the phrasing.
