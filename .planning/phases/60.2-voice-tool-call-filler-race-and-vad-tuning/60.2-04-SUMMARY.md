## Decisions Made

- **Prepend, don't replace** — the new paragraph was prepended BEFORE the pre-existing "TOOL NARRATION — CRITICAL RULE:" header content, not substituted for it. Gemini Live reads the system prompt linearly; front-loading the runtime-filler statement ensures it's the first thing the model encounters when composing a tool turn, maximizing adherence per the outcome-based prompt philosophy in MEMORY `feedback_livekit_prompt_philosophy.md`.
- **Preserve the filler examples** — CONTEXT explicitly authorized keeping them ("can stay as fallback prose but their criticality drops"). Removing them would have changed agent voice style on non-scoped-tool paths (no currently scoped non-runtime-filler tool exists, but future ones will exist and the tone shape matters). Labeled them "Fallback tone guidance (only if no runtime filler is playing for you)" so the model's mental model is: "when in doubt, these phrases match our brand."
- **Explicit prohibition wording** — used both "Do NOT speak your own filler phrase" AND "Do not generate your own filler text for these four" in different places. The Wave 0 test's assertion is a three-way OR, but using two distinct forms of the same instruction hardens adherence for Gemini (repetition at outcome-level is a known prompt lever for the 3.1 Flash Live model).
- **Outcome-framed, not directive-heavy** — per MEMORY feedback note, Gemini 3.1 Flash Live responds better to outcome framing than imperative directives. The opening paragraph leads with the outcome ("the two will overlap and cut each other off mid-sentence on the caller's line") before the prohibition, so the model understands WHY not just WHAT. Contrast with a bare "Do not speak filler" which reads as arbitrary.
- **Single-line 4-tool list** — rewrapped the Python string literal so `check_availability, book_appointment, capture_lead, or transfer_call` fits on one concatenated string line. The plan's grep acceptance criterion requires this verbatim phrase to match; line-wrapping broke it on the first pass, so I restructured the sentence to put the list inside parentheses on one line.
