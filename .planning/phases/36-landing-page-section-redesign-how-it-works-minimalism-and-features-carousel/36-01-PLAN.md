**Component structure:**
- Renders 4 steps, each as a `<div>` (not `<section>`) with `min-h-screen` and alternating backgrounds (per D-03): even index = `bg-white`, odd index = `bg-[#FAFAF9]`
- Each step uses `relative flex flex-col items-center justify-center min-h-screen py-16 md:py-24 lg:py-32 px-4 md:px-6 overflow-hidden`

**Per-step animations (per D-09):**
- Use 4 individual `useRef(null)` calls at top level (NOT inside .map) per Research Pitfall 6:
  ```jsx
  const step0Ref = useRef(null);
  const step1Ref = useRef(null);
  const step2Ref = useRef(null);
  const step3Ref = useRef(null);
  const stepRefs = [step0Ref, step1Ref, step2Ref, step3Ref];
  ```
- Each step gets its own `useInView(stepRefs[i], { once: true, margin: '-20% 0px -20% 0px' })` — also 4 individual calls at top level
- Each step gets its own `useScroll({ target: stepRefs[i], offset: ['start end', 'end start'] })` and `useTransform(scrollYProgress, [0, 1], [-12, 12])` for parallax — also 4 individual calls at top level

**Stagger fade-in (per D-09):**
- Use `motion.span` / `motion.div` with explicit `transition={{ delay: N }}` per element:
  - Step number: delay 0s
  - Icon container: delay 0.2s
  - Title: delay 0.4s
  - Gradient accent line: delay 0.6s (uses `scaleX: 0 -> 1` animation with `origin-left` per UI-SPEC)
  - Description: delay 0.8s
- Initial state: `{ opacity: 0, y: 20 }`, animated state: `{ opacity: 1, y: 0 }`
- Duration: 0.5s, easing: `[0.22, 1, 0.36, 1]`
- Do NOT use AnimatedStagger/AnimatedItem (50ms gap too small per Research)

**Parallax (per D-10):**
- Apply `useTransform`-driven `translateY` to the icon container only via `motion.div style={{ y: iconY }}`
- Disable on mobile: use a `useEffect` + `useState` to check `window.matchMedia('(max-width: 767px)')` and conditionally set parallax to 0 on mobile
- When `useReducedMotion()` returns true, skip all motion animations (return static positions)

**Soft background shapes (per D-11):**
- One centered circle per step: `absolute inset-0 flex items-center justify-center pointer-events-none` with `aria-hidden="true"`
- Circle: `w-[280px] h-[280px] md:w-[400px] md:h-[400px] rounded-full` with `step.shapeFill`

**Gradient accent line (per D-12):**
- Between title and description: `w-20 h-0.5 rounded-full bg-gradient-to-r from-[#F97316] to-[#FB923C]`
- Animated with `scaleX: 0 -> 1` and `origin-left` class, delay 0.6s

**Step number typography:**
- `text-[5rem] md:text-[6rem] lg:text-[8rem] font-semibold leading-none select-none pointer-events-none` with `step.numberColor`
- `aria-hidden="true"` on the step number span

**Icon container:**
- `w-16 h-16 md:w-20 md:h-20 lg:w-24 lg:h-24 rounded-3xl flex items-center justify-center` with `step.iconBg`
- Icon inside: `w-8 h-8 md:w-10 md:h-10 lg:w-12 lg:h-12` with `step.iconColor`

**Title:**
- `text-2xl md:text-3xl lg:text-5xl font-semibold tracking-tight leading-[1.2] text-[#0F172A]`

**Description:**
- `text-base md:text-lg leading-relaxed text-[#475569]`

**Accessibility:**
- Background shapes: `aria-hidden="true"` + `pointer-events-none`
- Step numbers: `aria-hidden="true"` + `select-none pointer-events-none`
- Icon containers: `aria-hidden="true"` (decorative)
- Gradient accent lines: `aria-hidden="true"`
- Titles are `<h3>` elements

**Performance (per D-26):**
- All animations are transform/opacity only
- No `position: sticky`
- No `backdrop-filter: blur()`
- Backgrounds are static CSS classes, not animated
