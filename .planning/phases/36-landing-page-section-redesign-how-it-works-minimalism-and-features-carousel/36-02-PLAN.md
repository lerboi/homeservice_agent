5. **scrollToIndex function:**
```jsx
const scrollToIndex = useCallback((idx) => {
  const clamped = ((idx % FEATURES.length) + FEATURES.length) % FEATURES.length;
  const track = trackRef.current;
  const cards = track?.children;
  if (!cards?.[clamped]) return;
  cards[clamped].scrollIntoView({ behavior: 'smooth', block: 'nearest', inline: 'center' });
  setActiveIndex(clamped);
}, []);
```

6. **IntersectionObserver for activeIndex sync on swipe** (per Research Pattern 3):
- In `useEffect`, create an IntersectionObserver with `root: trackRef.current`, `threshold: 0.5`
- Observe each card child element
- When a card crosses 50% visibility, update `activeIndex` to that card's index
- Cleanup observer on unmount

7. **Auto-advance** (per D-15, Research Pattern 4):
- `useRef` for intervalRef
- Auto-advance interval: 5000ms (5 seconds)
- `handleUserInteraction`: clears interval, sets 8000ms timeout to restart
- Arrow clicks, icon nav clicks, and scroll/swipe all call `handleUserInteraction`
- `useReducedMotion()` — if true, disable auto-advance entirely
- Cleanup interval on unmount
