**src/lib/payment-calculations.js** — Pure function for testability (matches Wave 0 test contract):

```javascript
/**
 * Calculate the new invoice status based on payment state.
 * Pure function — no side effects, no database calls.
 *
 * @param {Object} params
 * @param {number} params.invoiceTotal - Invoice total amount
 * @param {number} params.paymentsSum - Sum of all recorded payments
 * @param {string} params.currentStatus - Current invoice status
 * @param {string} params.dueDate - Due date (YYYY-MM-DD)
 * @param {string} params.today - Today's date (YYYY-MM-DD)
 * @returns {{ balance: number, newStatus: string }}
 */
export function calculatePaymentStatus({ invoiceTotal, paymentsSum, currentStatus, dueDate, today }) {
  // Do not auto-transition draft or void
  if (currentStatus === 'draft' || currentStatus === 'void') {
    return { balance: invoiceTotal - paymentsSum, newStatus: currentStatus };
  }

  const balance = Math.max(0, invoiceTotal - paymentsSum);

  if (balance <= 0) {
    return { balance: 0, newStatus: 'paid' };
  }

  if (paymentsSum > 0) {
    return { balance, newStatus: 'partially_paid' };
  }

  // No payments — check if overdue
  if (dueDate && today > dueDate) {
    return { balance, newStatus: 'overdue' };
  }

  return { balance, newStatus: 'sent' };
}
```

**Payments API** (`src/app/api/invoices/[id]/payments/route.js`):
Create with GET, POST, DELETE handlers. Use `createSupabaseServer()` + `getTenantId()` pattern.

**GET** — List payments for an invoice:
- Fetch payments ordered by payment_date DESC, created_at DESC
- Also fetch the invoice total
- Calculate: payments_sum = SUM(payments.amount), balance = invoice.total - payments_sum
- Return: `{ payments, payments_sum, balance, invoice_total }`

**POST** — Record a payment (per D-08):
Accept body: `{ amount, payment_date, note }`
- Validate: amount > 0, payment_date is a valid date
- Insert into `invoice_payments` with invoice_id from URL params, tenant_id
- After insert, recalculate using `calculatePaymentStatus`:
  - Fetch all payments for this invoice, sum amounts
  - Fetch invoice total and current status
  - Call `calculatePaymentStatus({ invoiceTotal, paymentsSum, currentStatus, dueDate, today })`
  - Update invoice status to newStatus if changed, set paid_at = now() if newly paid
- Return: `{ payment, payments_sum, balance, status }`

**DELETE** — Remove a payment:
Accept query param: `?payment_id=uuid`
- Delete from `invoice_payments` WHERE id = payment_id AND tenant_id
- After delete, recalculate using `calculatePaymentStatus` (same logic as POST)
- Return: `{ success: true, payments_sum, balance, status }`
