**Cron route** (`src/app/api/cron/invoice-reminders/route.js`):
GET handler following trial-reminders pattern exactly:

1. CRON_SECRET Bearer auth check
2. Use service-role supabase client (`import { supabase } from '@/lib/supabase'`)
3. Get today's date: `new Date().toISOString().split('T')[0]`

**Reminder dispatch:**
4. Query all invoices WHERE:
   - `status IN ('sent', 'overdue', 'partially_paid')`
   - `reminders_enabled = true`
   - `due_date IS NOT NULL`
   - Join with tenants to get owner info, and invoice_settings for business info
5. For each invoice, calculate which reminder points are due using `date-fns`:
   - `addDays(new Date(due_date), -3)` formatted === today → `'before_3'`
   - `due_date` === today → `'due_date'`
   - `addDays(new Date(due_date), 3)` formatted === today → `'overdue_3'`
   - `addDays(new Date(due_date), 7)` formatted === today → `'overdue_7'`
6. For each matching reminder point, check `invoice_reminders` table for existing record (UNIQUE on invoice_id + reminder_type prevents dupes)
7. If not yet sent: send email via `getResendClient()` using InvoiceReminderEmail template, send SMS via `getTwilioClient()` (non-fatal):
   - SMS body: `"{businessName}: Payment reminder — Invoice #{number} for ${amountDue} {statusPhrase}. Questions? Call {phone}"`
   - Status phrases per UI-SPEC: "is due on {date}" / "is due today" / "was due on {date}"
8. Insert into `invoice_reminders` (idempotency record)

**Late fee application** (runs in same cron):
9. Query all invoices WHERE:
   - `status IN ('overdue', 'partially_paid')` (only overdue invoices)
   - `due_date < today` (confirmed overdue)
   - Join with invoice_settings WHERE `late_fee_enabled = true`
10. For each qualifying invoice:
    - Check `late_fee_applied_at`:
      - If `late_fee_type = 'flat'` and `late_fee_applied_at` is NOT NULL → skip (flat fee applied once only)
      - If `late_fee_type = 'percentage'` and `late_fee_applied_at` is NOT NULL → check if 30+ days since last application. If < 30 days, skip.
    - Calculate late fee amount:
      - Flat: `settings.late_fee_amount`
      - Percentage: `invoice.total * settings.late_fee_amount / 100` (apply to original total, not compounded — get original total by subtracting existing late fee line items)
    - Calculate days overdue: `differenceInDays(today, due_date)` from date-fns
    - Insert line item into `invoice_line_items`:
      - `item_type = 'late_fee'`
      - `description = 'Late fee — {days} days overdue'`
      - `quantity = 1`
      - `unit_price = calculated_fee`
      - `taxable = false`
      - `line_total = calculated_fee`
    - Recalculate invoice totals: fetch all line items, use `calculateInvoiceTotals`, update invoice subtotal/tax_amount/total
    - Update invoice `late_fee_applied_at = now()`
11. Return `{ reminders_sent: N, late_fees_applied: M }`
