  <action>
Create `src/app/api/invoice-settings/route.js` with two handlers:

**GET handler:**
1. `const supabase = await createSupabaseServer()` and `const tenantId = await getTenantId()`
2. If no tenantId, return 401.
3. Query `invoice_settings` where `tenant_id = tenantId`.
4. If no row exists, auto-create one by:
   a. Querying `tenants` table for `business_name` and `owner_email` where `id = tenantId`
   b. Inserting a new `invoice_settings` row with `tenant_id = tenantId`, `business_name = tenant.business_name`, `email = tenant.owner_email`, all other fields at defaults (tax_rate: 0, payment_terms: 'Net 30', invoice_prefix: 'INV')
   c. Returning the newly created row
5. Return the settings row as JSON.

**PATCH handler:**
1. Auth check same as GET.
2. Parse request body. Allowed fields: `business_name`, `address`, `phone`, `email`, `logo_url`, `license_number`, `tax_rate`, `payment_terms`, `default_notes`, `invoice_prefix`.
3. Validate `tax_rate` is a number between 0 and 1 (e.g., 0.0825 for 8.25%). If invalid, return 400.
4. Validate `payment_terms` is one of: 'Net 15', 'Net 30', 'Net 45', 'Net 60'. If invalid, return 400.
5. Validate `invoice_prefix` is 1-10 alphanumeric characters. If invalid, return 400.
6. Update `invoice_settings` where `tenant_id = tenantId` with the provided fields + `updated_at = new Date().toISOString()`.
7. Return the updated row.

Import pattern:
```javascript
import { createSupabaseServer } from '@/lib/supabase-server';
import { getTenantId } from '@/lib/get-tenant-id';
```

A 'use client' page component with:

**Form sections** (each in a Card per UI-SPEC):

Section A — Business Identity:
- Business Name: `<Input>` bound to `settings.business_name`
- Address: `<Input>` (or textarea) bound to `settings.address`
- Phone: `<Input type="tel">` bound to `settings.phone`
- Email: `<Input type="email">` bound to `settings.email`
- License Number: `<Input>` bound to `settings.license_number`
- Logo Upload: A 96x96px preview area. If `settings.logo_url` exists, show the logo image. Else show a placeholder icon (ImagePlus from lucide). File input accepts PNG/JPG, max 2MB. On file select:

  b. Upload to Supabase Storage bucket `invoice-logos` at path `{tenantId}/logo.{ext}` with `upsert: true`

  d. Update `settings.logo_url` with the public URL
  e. Show toast: "Logo uploaded"
  Note: Use `supabase` from `@/lib/supabase-browser` for the client-side storage upload.
