## Interaction Contracts

### Hero Section

- **Spline 3D (desktop only, md+):** Cursor-reactive model loaded via dynamic import with spinner skeleton (aspect-[4/3], matches final dimensions for zero CLS). No Spline on `< md` — static image fallback (`/images/dashboard-mockup.png`) displayed.
- **RotatingText:** Cycle "Competitor / Revenue / Customer" at 3000ms interval. Static SSR fallback renders "Competitor" in accent color.
- **Primary CTA hover:** Existing pattern — `hover:bg-[#C2410C]/90`, `hover:shadow-[0_4px_16px_0_rgba(194,65,12,0.4)]`, `hover:-translate-y-0.5`. Duration 200ms ease.
- **Secondary CTA hover:** `hover:bg-white/[0.06]`. No translate.
- **Eyebrow pip:** `animate-pulse` — inherits Tailwind default pulse (2s).

### How It Works Section — Tab Redesign

- **Tab navigation:** Three numbered tabs `[01] [02] [03]` displayed horizontally at section top.
- **Active state:** Accent color underline or filled pill on active tab. Inactive tabs use muted text at 50% opacity.
- **Tab transition:** Fade + slight upward translate (8px) on exit, fade + downward from 8px on enter. Duration: 250ms ease-out. Honors `prefers-reduced-motion` (instant swap, no animation).
- **Panel content:** Single panel below tabs. Each step shows: step number badge, Lucide icon (Phone/Brain/CalendarCheck), title, description, italic detail quote.
- **Auto-advance:** Optional — tabs do NOT auto-advance (user controls progression; avoids distraction).
- **CTA below tabs:** "Start My 5-Minute Setup" in accent fill — same button style as existing.

### Features Bento Grid

- **Grid structure (md+):** 6-column grid. New 5-card layout:
  - Card 1 "The Night Shift, Sorted" — `col-span-4 row-span-2` (hero dark card)
  - Card 2 "The Golden Lead Filter" — `col-span-2`
  - Card 3 "Money in the Calendar" — `col-span-2`
  - Card 4 "Instant Emergency SMS" — `col-span-4`
  - Card 5 "Speaks Their Language" — `col-span-2` (light card, language flag treatment)
- **Hover micro-interaction:** `hover:-translate-y-0.5` (2px lift), shadow deepens. Duration 300ms ease. GPU-accelerated via `will-change: transform` (applied only on hover entry).
- **Hover glow:** `opacity-0 group-hover:opacity-100` gradient overlay — existing pattern, maintain.
- **Justification line:** Always visible (not hover-only) per existing implementation. Green dot + green text.
- **Card 5 treatment:** Light card (bg-white) with a subtle language-switcher visual — small flag icons or globe icon (Lucide Globe), body copy highlights multi-language capability.

### Social Proof Section

- **Card hover:** Upgrade from `hover:bg-white/[0.05] hover:border-white/[0.1]` to also include `hover:-translate-y-1` (4px lift). Duration 300ms ease.
- **Metric badge:** Retain green badge treatment. Consider subtle shimmer/glow on hover via box-shadow.
- **No carousel** — 3-card static grid maintained.

### Final CTA Section

- **Background:** Keep `bg-[#C2410C]` base. Upgrade radial gradient treatment — add a third gradient layer for depth.
- **Subtle animation:** Slow-moving radial gradient shift via CSS keyframes (10s infinite), opacity range 0.6–1.0. Must honor `prefers-reduced-motion` (static on reduced motion).
- **CTA button:** Inverted — dark button on copper background (existing AuthAwareCTA `variant="cta"` pattern).

## Section-by-Section Implementation Notes

### Section 1: Hero (D-01 through D-07)
- Layout: `md:grid md:grid-cols-12` — text col-span-6/lg:col-span-7, 3D col-span-6/lg:col-span-5
- Left: Eyebrow → H1 with RotatingText → subhead → CTA pair → avatar stack
- Right desktop: SplineScene (serafim/splite, cursor-reactive) in `aspect-[4/3]` container
- Right mobile: `<Image>` static fallback, `priority` attribute (above fold)
- Background treatment: `bg-[#0F172A]` + radial gradient accent + grid texture + floating orb — maintain existing

### Section 2: How It Works (D-08 through D-12)
- **Breaking change:** Replace stacked `AnimatedItem` cards with single tab panel interface
- Tab bar: `flex gap-2` row of 3 buttons — `[01]`, `[02]`, `[03]` labels
- Active tab: accent underline 2px or filled pill background at 10% accent opacity
- Panel: Single rounded card (`rounded-2xl`, `border border-black/[0.04]`, `bg-[#F5F5F4]/50`) — content transitions on tab change
- Each panel content: large step number (decorative, low-opacity), icon container (gradient bg from step data), title, description, italic detail
- Panel min-height: 280px (desktop) — prevents height jump between steps
- AnimatedSection wrapper: remove stagger, use single fade-in on initial mount only

### Section 3: Features Bento Grid (D-13 through D-15)
- Grid: `grid grid-cols-1 md:grid-cols-6 gap-3`
- 5 cards with spans as specified above
- Card 5 ("Speaks Their Language"): `col-span-2`, light card style (bg-white), Globe icon (Lucide), language badges as decorative pill elements (e.g., "ES", "EN", "SG" pill badges in muted tone)
- Section H2 update: "Five features. One question:" replacing current "Four features."
- Aceternity bento inspiration: cards should feel distinct in visual weight — hero card (dark) anchors grid, other cards support it

### Section 4: Social Proof (D-16 through D-17)
- Card hover: add `hover:-translate-y-1` to existing hover styles
- Metric badge on hover: `hover:shadow-[0_0_16px_rgba(22,101,52,0.2)]` — subtle green glow
- No content changes — testimonial copy locked

### Section 5: Final CTA (D-18 through D-19)
- Add third radial gradient: `bg-[radial-gradient(ellipse_at_center,rgba(255,255,255,0.05),transparent_40%)]`
- Keyframe animation for gradient breathing: `@keyframes cta-pulse { 0%, 100% { opacity: 0.6 } 50% { opacity: 1 } }` at 8s infinite — apply to top-left gradient layer only
- Reduced motion: skip animation, keep static gradient
