**Active nav link copper underline (D-09):**

For desktop page links (Pricing, About, Contact — NOT anchor links like #features, #how-it-works), add active link detection using the existing `pathname` from `usePathname()`.

Create a helper:
```jsx
const isActive = (href) => pathname === href || pathname.startsWith(href + '/');
```

For each page link, apply conditional classes:
- Active: `text-white relative` with a pseudo-element underline — use a `<span>` or `after:` for the copper underline to avoid layout shift:
```jsx
<Link href="/pricing" className={`text-sm transition-colors relative pb-1 ${
  isActive('/pricing')
    ? 'text-white'
    : 'text-white/60 hover:text-white'
}`}>
  Pricing
  {isActive('/pricing') && (
    <span className="absolute bottom-0 left-0 right-0 h-0.5 bg-[#C2410C]" />
  )}
</Link>
```

Apply this pattern to: `/pricing`, `/about`, `/contact`.
Anchor links (#features, #how-it-works) keep their existing `text-white/60 hover:text-white` style with NO active underline.
