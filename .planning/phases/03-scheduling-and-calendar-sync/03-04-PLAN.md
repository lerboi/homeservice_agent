    Create `src/app/api/working-hours/route.js`:
    - GET: Auth check via createSupabaseServer. Fetch tenant by owner_id = auth.uid(). Return `{ working_hours, slot_duration_mins, tenant_timezone }`.
    - PUT: Auth check. Receive `{ working_hours, slot_duration_mins, tenant_timezone }` in body. Update tenants table. Return updated data.
    - working_hours shape: `{ monday: { open: "08:00", close: "17:00", enabled: true, lunchStart: null, lunchEnd: null }, tuesday: { ... }, ... }`

    Create `src/app/api/zones/route.js`:
    - GET: Auth check. Fetch service_zones for tenant (ordered by created_at). Also fetch zone_travel_buffers for tenant. Return `{ zones, travelBuffers }`.
    - POST: Auth check. Receive `{ name, postal_codes }`. Check count < 5. Insert into service_zones. Return new zone.

    Create `src/app/api/zones/[id]/route.js`:
    - PUT: Auth check. Receive `{ name, postal_codes }`. Update service_zones row. Return updated zone.
    - DELETE: Auth check. Delete from service_zones (cascade deletes zone_travel_buffers). Return `{ deleted: true }`.

    Add to `src/app/api/zones/route.js`:
    - PUT (for travel buffers): Receive `{ buffers: [{ zone_a_id, zone_b_id, buffer_mins }] }`. Upsert into zone_travel_buffers. Return updated buffers.

    Create `src/app/api/calendar-sync/status/route.js`:
    - GET: Auth check. Fetch calendar_credentials for tenant where provider='google'. Return `{ connected, calendar_name, last_synced_at }` or `{ connected: false }`.

    Create `src/app/api/calendar-sync/disconnect/route.js`:
    - POST: Auth check. Import revokeAndDisconnect from google-calendar.js. Call it with tenant_id. Return `{ disconnected: true }`.

    All routes follow the pattern: auth check -> supabase operation -> Response.json(). Error handling: try/catch, return `{ error: message }` with 500 status on failure, 401 on auth failure.
